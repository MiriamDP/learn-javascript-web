Ejercicio 5 - Crear/Eliminar elementos de forma dinamica

En el siguiente ejercicio creamos y eliminaremos elementos del DOM de forma dinamica.

  • Elemento inicial 1
  • Elemento inicial 2

Para añadir un elemento al DOM primero creamos un elemento del tipo deseado con createElement. Una vez creado el elemento le podemos dar contenido con los metodos vistos en el ejercicio 4. Por ultimo este elemento sera hijo de algun otro elemento del DOM, bien del body, de un div, de un ul... asi que lo añadiremos como hijo con appendChild


Para eliminar el ultimo hijo de un elemento primero seleccionaremos ese hijo con lastElementChild y entonces lo eliminamos con removeChild


lastChild no es lo mismo que lastElementChild con lastElementChild obtenemos objetos de tipo Element mientras que con lastChild podemos obtener tambien elementos de tipo Text como pueden ser espacios, saltos o tabulaciones. Por ejemplo en las listas la tabulacion que tienen es un elemento de tipo Text, por lo que es importante tener en cuentas estas casuisticas a la hora de manipular el DOM
