Ejercicio 3 - querrySelecto y querrySelectorAll

Roses are red

violets are blue

my dog is my favorite

but you're OK too

Valentine's Day Card. Unknown
Aqui se mostraran los resultado de pulsar cada boton

Con querrySelector podemos seleccionar el primer elemento que tenga una cierta clase o etiqueta del DOM. Al pulsar el boton vemos que solo recuperamos el texto del primer elemento con la clase color


Si en su lugar usamos querrySelectorAlll seleccionamos todos los elementos del DOM con la clase o etiqueta elegida


Es importante tener en cuenta que al seleccionar clases o ids con cualquiera de estos metodos hay que especificarlos como en CSS


El objeto devuelto en el caso de querrySelectorAll es de tipo NodeList es por este motivo que al trabajar con este tipo de elementos podemos iterar sobre ellos con un bucle foreach. Podemos comprobar el tipo de elemento abriendo la consola del inspector
