Ejercicio - Trabajando con las cookies



El primer modo que tenemos para guardar informacion en el navegador son las cookies. Y aunque estas suelen ser más molestas que las de chocolate también tienen su utilidad


Las cookies no son mas que una tupla clave-valor. En una cookie almacenamos texto plano. En principio podemos almacenar cualquier clave que queramos aunque hay algunas reservadas con funcionalidad como por ejemplo expire, max-age o path


Dentro de una cookie podemos tener mas de una tupla. Para separarlas entre si usaremos ;


Normalmente las cookies suelen tener una caducidad, que es el tiempo durante el cual la cookie esta activa. Podemos especificar este tiempo con expire o con max-age. Con el primero especificamos en que fecha queremos que caduque y con el segundo cuanto tiempo pasara desde que se crea hasta que se caduque en segundos. Si no especificamos un max-age o un expire al crearla estaremos creando una cookie de sesion(sin caducidad) que estara activa hasta que cerremos el navegador o la destruyamos


Ademas es importante indicar el path que no es mas que el lugar donde se guarda la cookie. Por defecto el path es /


Para poder acceder a las cookies y manipularlas o borrarlas usaremos document.cookie. Cada cookie que se crea se almacena aqui. Las cookie caducadas no se borraran del fichero, pero si se borran manualmente si desapareceran al recargar la pagina


Para eliminar una cookie basta con hacer volver a crear la cookie con max-age=0 al actualizar la cookie


Tambien hay que tener en cuenta que para trabajar con cookies no todos los navegadores son tan permisivos. Para comprobar esta ejercicio te recomendamos que uses Firefox en lugar de Chrome
