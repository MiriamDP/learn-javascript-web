Ejercicio - Mostrar y ocultar elementos usando JS

Pulsa los botones para ocultar o mostrar el texto oculto

Hasta el infinito y más allá. Que la fuerza te acompañe

Con JS podemos cambiar los estilos de un elemento/s mediante la propiedad style. Por ejemplo en este caso estamos cambiando la proiedad display de none a block con el siguiente codigo
mensaje.style.display= "none"
para ocultarlo
