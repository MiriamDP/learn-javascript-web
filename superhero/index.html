Superhero

Por como esta configurada la API de Superhero tanto habiendola publicado en github pages como desde aqui siempre que intento hacer la llamada aparece el siguiente codigo de error Access to fetch at 'https://superheroapi.com/api/5827e69a09b59ce2426d3eda804acc0b/search/batman' from origin 'http://localhost:4300' has been blocked by CORS policy: No 'Access-Control-Allow-Origin' header is present on the requested resource.. Este error se produce por que para poder acceder a esta API se requiere hacer las peticiones desde un servidor y no las permite desde el front. Las posibles soluciones que he encontrado que se podrian implementar pasan por crear mi propio back para hacer esas peticiones pero eso queda fuera del alcance de esta asignatura por lo que no he podido continuar con el ejercicio propuesto para Superhero API
